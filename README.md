Selected Track

Multilingual Toxic Comment Classification (Advanced)

Problem Statement

This project builds a multi-label classification system to detect different types of toxic content such as insults, threats, and obscene language in multilingual text.

Approach & Methodology

- Text preprocessing (cleaning and normalization)
- Tokenization using BERT tokenizer
- Fine-tuning multilingual BERT model
- Multi-label classification using sigmoid activation
- Binary Cross Entropy loss

Tools & Technologies

- Python
- HuggingFace Transformers
- PyTorch
- Streamlit

How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py
